# 题目描述：
给你两个正整数a,b,  输出它们公约数的个数。

例如：a = 24， b = 36

则输出：6

# 分析
公约数，我就不是很喜欢 **约数** 这个概念。我喜欢用**因数**一点。

因数就是 如果 a % b == 0  那么 b 就是 a 的一个因数。 

这里选较小的一个数，可以少循环几次。

```python3
c = min(a, b)
res = 0
for i in range(1, c+1):
    if a % i == 0 and b % i == 0:
        res += 1

print(res)
```
