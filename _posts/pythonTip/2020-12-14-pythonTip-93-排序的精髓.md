**题目描述：**
给你一个整数list L,现在只允许你有一种操作：交换列表中相邻的两个元素。
对于一个给定的L，请你输出至少需要多少次交换可以让L变成升序序列。
如：L=[2, 8, 0, 3],则输出3。
注解: 2 8 0 3
swap (8 0)=> 2 0 8 3
swap (2 0)=> 0 2 8 3
swap (8 3)=> 0 2 3 8

**示例：**
输入：
L = [2, 8, 0, 3]
输出：
3
